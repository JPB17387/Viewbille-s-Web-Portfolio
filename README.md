# Viewbille Mae Empis - Personal Portfolio

![Viewbille Portfolio Preview](profile.png)

Welcome to the personal portfolio of **Viewbille Mae Empis**. This project is a customized, responsive, and dynamic web portfolio built to showcase web development skills, projects, and personal information in a clean, modern interface.

## About This Project

This portfolio is developed with a focus on building clean, functional, and responsive web applications. The goal of this project is to present a professional online presence, integrating modern web aesthetics and smooth navigation interactions. 

It serves as a testament to my development philosophy: *"Code. Build. Learn. Repeat."*

## Features

- **Responsive Design**: Ensuring accessibility across mobile, tablet, and desktop views.
- **Dynamic Content**: JavaScript handled navigation between different sections (About, Projects, Contact, About the Dev) without page reloads.
- **Clean UI/UX**: Utilizing a modern color palette, custom typography (Montserrat and Playfair Display), and styled sections for an engaging user experience.
- **Smooth Animations**: Incorporating CSS transitions and interactive button elements that provide a premium feel.

## Built With

- **HTML5**: Semantic structuring
- **CSS3** (Vanilla): Custom styling, variables, layout, and responsive media queries
- **JavaScript** (Vanilla): DOM manipulation and interactivity

## Project Structure

```bash
├── viewbille.html     # Main HTML document
├── style.css          # Styling and layout
├── functionality.js   # Interactivity and dynamic navigation
├── profile.png        # Profile imagery
└── Viewbille.jpg      # Background/Preview imagery
```

## Getting Started

To view this portfolio locally:

1. Clone this repository or download the source code.
2. Navigate to the project directory.
3. Simply double-click on `viewbille.html` to open it in your default web browser. No server setup, Node.js, or package manager is required.
